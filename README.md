# 🚑 LifeLink AI

An AI-powered digital ambulance management system built using the MERN stack.

## 🚀 Features

- User Authentication (JWT)
- Emergency SOS Requests
- Driver Dashboard
- Hospital Dashboard
- Live Ambulance Tracking
- AI Severity Prediction
- Real-time Notifications

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt
